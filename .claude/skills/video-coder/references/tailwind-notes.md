# Arbitrary Values (Quick Custom Values)

Arbitrary values let you insert **any exact CSS value** inside a Tailwind class using **square brackets**, giving you full freedom beyond Tailwindâ€™s default scales.

## What They Do
Let you set **custom sizes, spacing, colors, borders, radii, typography, and positioning** instantly.

## Examples
- `w-[37px]`
- `h-[3.5rem]`
- `p-[18px]`
- `m-[2.75rem]`
- `bg-[#1a73e8]`
- `text-[22px]`
- `border-[3px]`
- `rounded-[14px]`
- `gap-[22px]`
- `top-[42px]`
- `z-[25]`

## When to Use
Use arbitrary values for **one-off, precise custom values** without editing your Tailwind config.
